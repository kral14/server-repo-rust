# MasterDeploy

MasterDeploy layihəsini serverinizdə və ya lokal kompüterinizdə avtomatik qurmaq üçün aşağıdakı qısa əmrlərdən istifadə edə bilərsiniz.

## Linux və Mac üçün
Terminalda bu əmri işə salın:
```bash
curl -sL https://raw.githubusercontent.com/kral14/server-repo-rust/main/remote_installer.py | python3 -
```

## Windows üçün
PowerShell-i açıb bu əmri daxil edin:
```powershell
irm https://raw.githubusercontent.com/kral14/server-repo-rust/main/install.ps1 | iex
```
