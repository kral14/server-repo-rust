📖 Cloudflare Worker & Tünel Quraşdırma Təlimatı

Addım-addım tünelin sabit linkə bağlanması

1. Cloudflare KV Yaradılması:

Cloudflare panelində Workers & Pages -> KV -> Create Namespace bölməsinə daxil olub TUNNEL_DB adında bir verilənlər bazası (KV) yaradın və ID-sini kopyalayıb panelimizin sazlamalarına yazın.

2. Cloudflare Worker Yaradılması:

Workers & Pages -> Overview -> Create Application bölməsindən yeni Worker yaradın (məs: yeni-test) və aşağıdakı kodu kopyalayaraq daxilinə yapışdırın:

export default {
  async fetch(request, env, ctx) {
    // Layihə adı DƏQİQ MasterDeploy panelindəki ilə eyni olmalıdır
    const APP_NAME = "yeni-test"; 

    // KV-dən canlı linki oxuyuruq
    const liveUrl = await env.TUNNEL_DB.get(APP_NAME);

    if (!liveUrl) {
      return new Response(`[MasterDeploy] '${APP_NAME}' üçün aktiv tünel tapılmadı. Zəhmət olmasa panelinizdən tüneli başladın.`, {
        status: 503,
        headers: { "Content-Type": "text/plain; charset=UTF-8" }
      });
    }

    // Sorğunu canlı linkə yönləndiririk
    const url = new URL(request.url);
    const targetUrl = url.href.replace(url.origin, liveUrl.trim());

    const modifiedRequest = new Request(targetUrl, {
      method: request.method,
      headers: request.headers,
      body: request.body,
      redirect: 'manual'
    });

    return fetch(modifiedRequest);
  }
}
3. KV Binding (Ən Vacib Addım):

Yaratdığınız Worker-in Settings -> Variables -> KV Namespace Bindings bölməsinə daxil olun. Add Binding deyərək Variable Name hissəsinə DƏQİQ TUNNEL_DB yazın, KV Namespace olaraq isə yaratdığınız TUNNEL_DB bazasını seçib yadda saxlayın.

4. Tüneli Başlatmaq:

Panelimizdən layihə üçün Tünel Al düyməsini sıxaraq tüneli başladın. 5-10 saniyə sonra Worker linkinizə daxil olduqda saytınız artıq internetdə canlı olaraq açılacaqdır!
